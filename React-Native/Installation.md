To create a new Expo project:
Website: https://reactnative.dev/

```bash
npx create-expo-app@latest
```

To run Project:

```bash
npx start expo
```

If you want to reset the project with minimum file:

```bash
npm run reset-project
```

## Installation with blank template

To create a new Expo project

```bash
npx create-expo-app YourAppName --template blank@latest
```
==`YourAppName` is your project folder name.==

Or add typescript
```bash
npx create-expo-app my-app --template
```
