#### Navigation between screens

`app/index.tsx` File Structure.
- How to add link to navigate to `second.tsx` file.

```tsx
import { AppText } from "@/components/AppText";
import { Link } from "expo-router";
import { View } from "react-native";

export default function IndexScreen(){
   return (
   <View>
      <Text>Index Screen</Text>
      <Link href="/second" push>
      Push to /second
      </Link>
   </View>
   )
}
```

- You can also used **Button** inner link.
    - If you use custom **Button** component or any other you need to pass `asChild` to there parent component.

```tsx
import { AppText } from "@/components/AppText";
import { Link } from "expo-router";
import { View } from "react-native";

export default function IndexScreen(){
   return (
   <View>
      <Text>Index Screen</Text>
      <Link href="/second" push asChild>
      <Button title="Push to /second" />
      </Link>
   </View>
   )
}
```

- Other way of navigate with **Expo Route**.
    - You can also used **Custom Button**.

```tsx
import { AppText } from "@/components/AppText";
import { Link, useRouter } from "expo-router";
import { View, Button } from "react-native";

export default function IndexScreen(){

const route = useRouter();

   return (
   <View>
      <Text>Index Screen</Text>
      <Button title="Push to /second"
              onPress={() => {
              router.push("/second")
              }} 
        />
   </View>
   )
}
```

#### Routes and folder

If you want to know how many routes in your application so run below command in terminal.

```bash
npx expo-route-sitemap
```

- `/app`
    - `(anything)`
        - `(fourth)`
            - `fifth`
                - `sixth`
                    - `index.tsx`

If you add nested routes So just pass the link below.
- They automatically show index page you not need to add on link.

```tsx
import { AppText } from "@/components/AppText";
import { Link } from "expo-router";
import { View } from "react-native";

export default function IndexScreen(){
   return (
   <View>
      <Text>Index Screen</Text>
      <Link href="/fifth/sixth" push asChild>
      <Button title="Push to /fifth/sixth" />
      </Link>
   </View>
   )
}
```

#### Screens and layout files

If you add `_layout.tsx` file in sixth folder and pass text so they show **layout** folder component not **Index**. You need to pass any navigator on layout like stack or slot [[#Navigators.]]

- `_layout.tsx` File.

```tsx
import { Stack } from "expo-router";

export default function Layout(){
   return <Stack />
}
```

--------------------------------------------------------------------------
#### layout file execution order

Add `_layout.tsx` File in fifth folder
- If we add **Slot** in it so they open **Sixth Index**.

```tsx
import { Slot } from "expo-router";

export default function Layout(){
   return <Slot />
}
```

- But If you pass any **Text** or `View` They show **Fifth Layout** file.

```tsx
import { View, Text } from "react-native";

export default function Layout(){
   return (
    <View>
        <Text>Fifth Screen Layout</Text>
    </View>)
}
```
