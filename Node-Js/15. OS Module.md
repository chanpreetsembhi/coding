
1. `userInfo()` This gave you information of your system.

```js
import os from 'os';

console.log(os.userInfor());
```

2. `totalmem()` This will return you total memory in bytes.

```js
import os from 'os';

console.log(os.totalmem());
```

3. `freemem()` This will return you free memory in bytes.

```js
import os from 'os';

console.log(os.freemem());
```

4. `cpus()` This will gave you every CPU core information.

```js
import os from 'os';

console.log(os.cpus());
```
