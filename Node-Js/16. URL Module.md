1. This will return you URL all information into a object.

```js
import url from 'url';

const urlString = "https://www.google.com/search/?q=hello+world";

const urlObj = new URL(urlString);

console.log(urlObj);
```

2. `format()` This will return you URL in exact way with format view not return you object.

```js
import url from 'url';

const urlString = "https://www.google.com/search/?q=hello+world";

const urlObj = new URL(urlString);

console.log(url.format(urlObj));
```

3. `import.meta.url` - This will return you file URL.

```js
import url from 'url';

console.log(import.meta.url);
```

4. `fileUrlToPath()` This will return you exact file path with file type.

```js
import url from 'url';

console.log(url.fileURLToPath(import.meta.url));
```

5. `URLSearchParams()` 
   - `params.get("q")` return you hello world.
   - `params.append("limit", "5");` add values.
   - `params.delete('limit')` delete the value.
```js
import url from 'url';

const urlString = "https://www.google.com/search/?q=hello+world";

const urlObj = new URL(urlString);

const params = new URLSearchParams(urlObj.search);

params.append("limit", "5");

params.delete('limit')

console.log(params.get("q"));
```
