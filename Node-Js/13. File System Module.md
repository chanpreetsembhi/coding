#### How to Read File.

Create a `test.txt` File.

```txt
This is from the text file
```

Create `fsDemo.js` File or anyone which name you want. There are some methods.

1. `readFile()` - callback method.

```js
import fs from 'fs';

fs.readFile('./test.txt', 'utf-8', (err, data) => {
    if (err) throw err;
    console.log(data);
})
```

2. `readFileSync()` - Synchronous version method.

```js
import fs from 'fs';

const data = fs.readFileSync('./test.txt', 'utf-8');
console.log(data);
```
==**Note:** Prefer this method.==

3. `readFile()` - `Promise`  `.then()` methods.

```js
import fs from 'fs/promises';

fs.readFile('./test.txt', 'utf-8')
    .then(data => console.log(data))
    .catch(err => console.log(err));
```

4. `readFile()` - async/await method.

```js
import fs from 'fs/promises';

const readFile = async () => {
    try {
        const data = await fs.readFile('./test.txt', 'utf-8');
        console.log(data);
    } catch (error) {
        console.log(error);
    }
}

readFile();
```
==**Note:** This is much more readable code.==

```js
    } catch (error) {

    }
}
```

#### How to Append Text in File.

`fs.demo.js` File Structure.

```js
const appendFile = async() => {
    try {
        await fs.appendFile('./test.txt', '\nThis is appended text');
        console.log('File is appended to...');
    } catch (error) {
        console.log(error);
    }
}
```
==**\n** mean write in next line.==
**NOTE:** You can Append Text in File doing also that all methods which describe in read file in top
