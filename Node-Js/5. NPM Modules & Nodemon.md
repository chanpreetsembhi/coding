This Package help you to run as a server it reload automatically restarting service while doing any changes in your file..
[Nodemon Official npm package](https://www.npmjs.com/package/nodemon)
#### Install Nodemon Through npm.

```
npm i nodemon
```

==Or if you want install only dependencies. Prefer this install.==

```
npm i -D nodemon
```

After installing you get One file `node_modules` and one file `package-lock.json` and **nodeman** added in **"devDependencies"** in your `package.json` file.

`package.json` File Structure.

```json
{
  "name": "js",
  "version": "1.0.0",
  "description": "Node.js crash course code",
  "main": "index.js",
  "type": "module",
  "scripts": {
    "start": "nodemon server.js"
  },
  "author": "Chanpreet Singh",
  "license": "MIT",
  "devDependencies": {
    "nodemon": "^3.1.4"
  }
}
```
