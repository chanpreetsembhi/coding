This will provide crypto graphic functionality.

1. `createHash()` will generate Bytes.

```js
import crypto from 'crypto';

const hash = crypto.createHash('sha256');

hash.update('password1234');

console.log(hash.digest('hex'));
```

2. `randomBytes()` will generate random Bytes in every time.

```js
import crypto from 'crypto';

crypto.randomBytes(16, (err, buff) => {
    if (err) {
        throw err;
    }
    console.log(buff.toString('hex'));
})
```

3. `createCipheriv()` & `createDecipheriv()` methods.

```js
import crypto from 'crypto';

const algorithem = 'aes-256-cbc';
const key = crypto.randomBytes(32);
const iv = crypto.randomBytes(16);

/* Cipheriv */
const cipher = crypto.createCipheriv(algorithem, key, iv);
let encrypted = cipher.update("Hello this is a secret message", "utf-8", "hex");
encrypted += cipher.final('hex');
console.log(encrypted);

/* Decipheriv */
const decipher = crypto.createDecipheriv(algorithem, key, iv);
let dencrypted = decipher.update(encrypted, "hex", "utf-8");
dencrypted += decipher.final('utf-8');
console.log(dencrypted);
```
