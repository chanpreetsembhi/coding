1. Create .env file.

```env
PORT=8000
```
==This also include in .gitignore file==

2. `package.json` file structure.

```json
{
"scripts": {
    "start": "nodemon --env-file=.env server.js"
  },
}
```
==**--env-file=** is declare file code .**env** is file name==

3. `server.js` File Structure.

```js
const PORT = process.env.PORT;
```
==**process.env** is way to get **.env** files variable in any JS file **PORT** is variable name==
