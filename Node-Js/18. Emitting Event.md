1. This is great for building real time applications. you can create Custom emits also.
    - You can register emit in `.on()` method.
    - You can emit event in `.emit()` method.

```js
import { EventEmitter } from "events";

const myEmitter = new EventEmitter();

function greethandler() {
  console.log("Hello World!");
}

function goodbyehandler() {
  console.log("Goodbye World!");
}

/* Register event listeners */
myEmitter.on("greet", greethandler);
myEmitter.on("goodbye", goodbyehandler);

/* Event Emittes */
myEmitter.emit("greet");
myEmitter.emit("goodbye");
```

2. Error Handlers.
    - You can register emit error in `.on()` method.
    - You can simulate or run error in `.emit()` method.
```js
import { EventEmitter } from "events";

const myEmitter = new EventEmitter();

myEmitter.on('error', (err) => {
    console.log("An Error", err);
});

/* simulate error */
myEmitter.emit('error', new Error('Something went wrong'));
```

3. Add values in handlers.

```js
import { EventEmitter } from "events";

const myEmitter = new EventEmitter();

function greethandler(name) {
  console.log("Hello World!" + name);
}

function goodbyehandler(name) {
  console.log("Goodbye World!" + name);
}

/* Register event listeners */
myEmitter.on("greet", greethandler);
myEmitter.on("goodbye", goodbyehandler);

/* Event Emittes */
myEmitter.emit("greet", "Singh");
myEmitter.emit("goodbye", "Singh");
```
