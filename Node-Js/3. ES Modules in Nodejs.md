#### How to use `ESModules`

1. if you want to use` ESModules` so add a **"type"** `"commonjs"` or `"module"` in `package.js`.

```json
{
  "name": "js",
  "version": "1.0.0",
  "description": "Node.js crash course code",
  "main": "index.js",
  "type": "module"
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "author": "Chanpreet Singh",
  "license": "MIT"
}
```

2. Create `index.js` and `posts.js` or any file name.
   - `post.js` File Structure.

```js
const posts = [
    { id: 1, title: 'Part One' },
    { id: 2, title: 'Part Two' }
];

const getPosts = () => posts;

// export const getPosts = () => posts;
// export { getPosts };
export default getPosts;
```

  - `index.js` File Structure.

```js
import getPosts, { getPostsLength } from "./postController.js";

console.log(getPosts());
console.log(`Post Lenght : ${getPostsLength()}`);
```
==**Note**: If you use **export default file name** Not use curly braces **{}** while import and rest of the methods curly braces **{}** use compulsory.==
