1. `process` is global so if you like to import you can until it works without import also.

```js
console.log(process.argv);
```

2. How to add value in the object.

```js
/* write this in terminal */
filename import anytext

console.log(process.argv[3]);
```

3. `process.env` method show all information.

```js
console.log(process.env.COMPUTERNAME);
```
==you can access all values with . method==

4. `.pid`

```js
console.log(process.pid);
```

5. `cwd()` method show you current folder location with folder name.

```js
console.log(process.cwd());
```

6. `title` method.

```js
console.log(process.title);
```

7. `memoryUsage()` method show usage in object.

```js
console.log(process.memoryUsage());
```

8. `update()` method code update time.

```js
console.log(process.uptime());
```

9. `exit()` method to exit the code after that no code work's.

```js
process.on('exit', (code) => {
    console.log(`About to exit with code ${code}`);
});
```
