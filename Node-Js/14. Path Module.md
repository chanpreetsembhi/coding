#### `ESModules` Path.

1. `basename()` This will return the file name.

```js
import path from "path";

const filePath = "./dir1/dir2/test.txt";

console.log(path.basename(filePath));
```

- This will return.

```
test.txt
```

--------------------------------------------------------------------------

2. `dirname()` This will return directory name.

```js
import path from "path";

const filePath = "./dir1/dir2/test.txt";

console.log(path.dirname(filePath));
```

- This will return.

```
./dir1/dir2
```

--------------------------------------------------------------------------

3. `extname()` This will return file type or file extension.

```js
import path from "path";

const filePath = "./dir1/dir2/test.txt";

console.log(path.extname(filePath));
```

- This will return.

```
.txt
```

--------------------------------------------------------------------------

4. `parse()` This will return all `root`, `dir`, `base,ext`, `name` in a single object.

```js
import path from "path";

const filePath = "./dir1/dir2/test.txt";

console.log(path.parse(filePath));
```

- This will return.

```
.txt
```

--------------------------------------------------------------------------
#### `CommonJs` Path.

```js
import url from 'url';

const filePath = "./dir1/dir2/test.txt";

const __filename = url.fileURLToPath(import.meta.url);
const __dirname = path.dirname(__filename);

console.log(__filename, __dirname);
```

- This will return.
- Your full directory with file name and directory with foldername

```
D:\Projects\node-js\pathDemo.js  D:\Projects\node-js
```

--------------------------------------------------------------------------

5. `join()` This will join files with directory until you are Linux or windows

```js
import path from 'path';
import url from 'url';

const __filename = url.fileURLToPath(import.meta.url);
const __dirname = path.dirname(__filename);

const filePath = path.join(__dirname, "dir1", "dir2", "text.txt");
console.log(filePath);
```

- This will return.

```
D:\Projects\node-js\dir1\dir2\text.txt
```

--------------------------------------------------------------------------

6. `resolve()` This is also similar or same like `join()`.

```js
import path from 'path';
import url from 'url';

const __filename = url.fileURLToPath(import.meta.url);
const __dirname = path.dirname(__filename);

const filePath = path.resolve(__dirname, "dir1", "dir2", "text.txt");
console.log(filePath);
```

- This will return.

```
D:\Projects\node-js\dir1\dir2\text.txt
```
