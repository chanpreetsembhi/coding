#### Create Routing.
- `app.js` File Structure.

```js
import express from "express";

const app = express()
  
/* Config Ejs */
app.set("view engine", "ejs");

/* first view is es engine and second is folder name */
app.set("views", "views");

app.get("/", (req, res) => {
  res.render("index", {
    title: "Welcome",
    message: "Hello from EJS",
    people: ["John", "James", "Selena"],
  });
});

app.listen(8000, () => console.log("Server Started"));
```

- Create a **views** Folder inner create a file `index.ejs`.

```ejs
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>
        <%= title %>
    </title>
</head>
<body>
        <h1>
            <%= message %>
        </h1>
</body>
</html>
```

#### Lopping in array.

```ejs
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>
        <%= title %>
    </title>
</head>
<body>
        <h1>
            <%= message %>
        </h1>
        
       <ul>
            <% people.forEach(person=> { %>
                <li>
                    <%= person %>
                </li>
                <% }) %>
        </ul>
</body>
</html>
```

#### Include File that can use in all pages.
- Create a `header.ejs` File.

```ejs
<header>
    <h1>Header</h1>
</header>
```

- Include in `index.ejs` file.

```ejs
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>
        <%= title %>
    </title>
</head>
<body>

    <%- include('partials/header.ejs') %>
        <h1>
            <%= message %>
        </h1>
        
</body>
</html>
```
