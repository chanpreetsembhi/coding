Using template engines with Express.

Create a `pakage.json` file

```txt
npm init
```

Install ES Template Engine with Express.

```txt
npm i express ejs
```

Change `package.json` to restarting server automatically.

```json
{
  "name": "express-ejs",
  "version": "1.0.0",
  "main": "index.js",
  "type": "module",
  "scripts": {
    "test": "node app",
    "dev": "node --watch app"
  },
  "keywords": [],
  "author": "",
  "license": "ISC",
  "description": "",
  "dependencies": {
    "ejs": "^3.1.10",
    "express": "^4.19.2"
  }
}
```

Create `app.js` File.

```js
import express from "express";

const app = express();

/* Config Ejs */
app.set("view engine", "ejs");

app.listen(8000, () => console.log("Server Started"));
```
