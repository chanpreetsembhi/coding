Like In **Node JS** we need to install `Nodemon` to restart server but In **Express JS** we not need any NPM packages. we just write script in `package.json`.

1. `package.json` file structure.

```json
"scripts": {
    "start": "node server",
    "dev": "node --watch server"
  },
```

Run the server.

```
npm run dev
```

If you want to add specific port so use it in .env file also.
1. `.env` file

```env
PORT=8000
```

2. `package.json` file.

```json
"scripts": {
    "start": "node server",
    "test": "node --watch --env-file=.env server"
  },
```
