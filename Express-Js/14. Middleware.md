1. Middleware basic syntax.

```js
const middleware = (req, res, next) => {
    (here your code);
    next();
}
```

#### Example Create a Middleware function which show your http path.

1. If you pass middle ware in one request.
   - `posts.js` File structure. pass middleware only get posts method.
   
```js
const logger = (req, res, next) => {
  console.log(
    `${req.method} ${req.protocol}://${req.get("host")}${req.originalUrl}`
  );
  next();
};

/* Get All Posts */
router.get("/", logger, (req, res) => {
  const limit = parseInt(req.query.limit);

  if (!isNaN(limit) && limit > 0) {
    return res.status(200).json(posts.slice(0, limit));
  }
  res.status(200).json(posts);
});
```

2. If you pass Middleware in app level.
   - Create a folder name middleware and inner create a file `logger.js` or any other which you want.

```js
const logger = (req, res, next) => {
  console.log(
    `${req.method} ${req.protocol}://${req.get("host")}${req.originalUrl}`
  );
  next();
};

export default logger;
```

   - `server.js` File Structure.

```js
import express from "express";

import posts from "./routes/posts.js";

import logger from "./middleware/logger.js";

const port = process.env.PORT || 8000;

const app = express();

/* Body parser middleware */
app.use(express.json());
app.use(express.urlencoded({ extended: false }));

/* logger middleware */
app.use(logger);

/* Routes */
app.use("/api/posts", posts);

app.listen(port, () => console.log(`Server is running on port ${port}`));
```
