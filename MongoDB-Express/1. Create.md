#### How to setup MongoDB with Express JS.
-  Create a `package.json` file with `npm init`

```
npm init
```
==They ask you some package related options==

Or skip question.

```
npm init -y
```

#### Install Express JS.

If you want to read more go to the Official [Express JS](https://expressjs.com/) website.

```
npm install express
```

#### Install MongoDB.
- If You want to connect MongoDB with Express JS you should install [`mongoosejs`](https://www.npmjs.com/package/mongoose) NPM Package.

```
npm install mongoose
```
