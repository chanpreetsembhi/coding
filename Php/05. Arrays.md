How to create arrays with key value pairs

```php
<?php
    $books = [
        [
            "name" => "Do Andriods Dream of Electric Sheep",
            "auther" => "Philip K. Dick",
            "releaseYear" => 2010,
            "purchaseUrl" => "https//example.com"
        ],
        [
            "name" => "The Langoliers",
            "auther" => "Andy Weid",
            "releaseYear" => 2006,
            "purchaseUrl" => "https//example.com"
        ]
    ]
?>

 <ul>
        <?php foreach ($books as $book): ?>
            <li>
                <a href="<?= $book['purchaseUrl'] ?>">
                    <?= $book['name'] ?> (<?= $book['releaseYear'] ?>)
                </a>
            </li>
        <?php endforeach; ?>
 </ul>
```
