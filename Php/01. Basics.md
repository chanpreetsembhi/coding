(3) How to get all information

```php
var_dump($_SERVER);

/* This will format all data readablilty */
echo <pre>;
var_dump($_SERVER);
echo </pre>;
```
`$_SERVER` Through we get all values like pages link etc. 

------------------------------------------------------------------

(4) How to kill the Execution. after that tag no PHP script will run

```php
die();

var_dump($_SERVER);
```
`$-SERVER` in not run because we declare `die()` function it kill all this after it.

------------------------------------------------------------------

(5) How to get or find page URI

```php
require($_SERVER['REQUEST_URI']);
```
use this in exact page like `home.php`, `about.php` etc.
