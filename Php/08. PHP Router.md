Create Folder `controllers` and paste `index.php`, `about.php`, `contact.php` in this folder.

Create `index.php` file in root folder.

```php
<?php
require "functions.php";

$uri = $_SERVER['REQUEST_URI'];

if ($uri === "/") {
    require "controllers/index.php";
} else if ($uri === "/about") {
    require "controllers/about.php";
} else if ($uri === "/contact") {
    require "controllers/contact.php";
} else {
    http_response_code(404);
    echo "404 Not Found";
}
```

To Run router create `.htaccess` file in root folder with inner content.

```
RewriteEngine On

# Redirect requests to index.php if file or directory doesn't exist
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^ index.php [QSA,L]
```

2. Second Simple method to create routers.

```php
<?php
require "functions.php";

$uri = parse_url($_SERVER['REQUEST_URI'])['path'];

$routes = [
    "/" => "controllers/index.php",
    "/about" => "controllers/about.php",
    "/contact" => "controllers/contact.php",
];

if (array_key_exists($uri, $routes)) {
    require $routes[$uri];
} else {
    http_response_code(404);
    require "views/404.php";
    
    die();
}
```

3. Code in small pieces of functions with reusable.

```php
<?php
require "functions.php";

$uri = parse_url($_SERVER['REQUEST_URI'])['path'];

$routes = [
    "/" => "controllers/index.php",
    "/about" => "controllers/about.php",
    "/contact" => "controllers/contact.php",
];

function routeToController($uri, $routes)
{
    if (array_key_exists($uri, $routes)) {
        require $routes[$uri];
    } else {
        abort();
    }
}

function abort($code = 404)
{
    http_response_code($code);
    require "views/{$code}.php";
    die();
}

routeToController($uri, $routes);
```
