How to Separate PHP logic and include with HTML to display the data.

##### Create a `separate-logic.php` file

```php
<?php
$books = [
    [
        "name" => "Do Andriods Dream of Electric Sheep",
        "author" => "Philip K. Dick",
        "releaseYear" => 2010,
        "purchaseUrl" => "https//example.com"
    ],
    [
        "name" => "The Langoliers",
        "author" => "Andy Weid",
        "releaseYear" => 2006,
        "purchaseUrl" => "https//example.com"
    ],
    [
        "name" => "The Art",
        "author" => "Andy Weid",
        "releaseYear" => 2016,
        "purchaseUrl" => "https//example.com"
    ]
];

$filteredBooks = array_filter($books, function ($book) {
    return $book['releaseYear'] > 2008;
});

require "separate-view.php";
```

###### Create a `seprate-view.php` or `seprate.view.php` file

```php
<ul>
        <?php foreach ($filteredBooks as $book): ?>
            <li>
                <a href="<?= $book['purchaseUrl'] ?>">
                    <?= $book['name'] ?> (<?= $book['releaseYear'] ?>) -                      By <?= $book['author'] ?>
                </a>
            </li>
        <?php endforeach; ?>
</ul>
```
