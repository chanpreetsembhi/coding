(1) How to assign value in variables

```php
<? php
$greeting = "Hello";

echo "$greeting EveryOne!"
?>
```

------------------------------------------------------------------

(2) How to add string into variable

```php
<?php
$greeting = "Hello"

echo $greeting . " " . "Everyone!";
?>
```

------------------------------------------------------------------

(3) how to declare variable in Html Code

```php
$message = "Hello World";

<h1><?php echo $message; ?></h1>

// short syntax
<h1><?= $message ?></h1>
```
