How to loop through arrays

```php
<?php
$books = [
        "Do Andriods Dream of Electric Sheep",
        "The Langoliers",
        "Hail Mary"
    ];
?>

<ul>
        <?php foreach ($books as $book) {
            echo "<li>$book</li>";
        } ?>
        <br />
        
        /* Or */ 
        <?php foreach ($books as $book): ?>
            <li><?= $book ?></li>
        <?php endforeach; ?>
    </ul>
```
