##### Create `home.php` , `about.php`, `contact.php` files

```php
 <?php
     $heading = "Home";
     require ('views/home.view.php')
```
==require or include help to get that component in file.==
create a variable and assign a value add this in all three home, about, contact pages with change text it will be dynamic.

##### Create a separate `nav.php` file

```php
<nav>
<a href="/">HomePage</a>
<a href="/about">About Us</a>
<a href="/contact">Contact Us</a>
</nav>
```


```php
<?php require('nav.php') ?>
```

#### How to active navbar link based on pages dynamic links

```php
<nav>
<a href="/" class="<?php if($_SERVER['REQUEST_URI'] === '/home.php'){
echo 'bg-gray-900 text-white'
} else{
'text-gray-300'
} ?> rounded-md px-3 py-2 text-sm">HomePage</a>
<a href="/about">About Us</a>
<a href="/contact">Contact Us</a>
</nav>
```
==use if-else condition to activate and deactivate tag based on conditions==

###### You can also use a Ternary operator while using if-else

```php
<nav>
<a href="/" class="<?php echo $_SERVER['REQUEST_URI'] === '/home.php' ? 
'bg-gray-900 text-white' : 'text-gray-300' ?> rounded-md px-3 py-2 text-sm">HomePage</a>
<a href="/about">About Us</a>
<a href="/contact">Contact Us</a>
</nav>
```
Ternary operator `condotion ? exprIfTrue : exprIfFalse`.

###### Create this active classes much more Cleaner.

-  Create a `function.php` file
```php
function urlIs($value){
     return $_SERVER['REQUEST_URI'] === $value;
}
```
==This function help to find page URL link if you want to use everywhere so create separate file and use where you use it.== 

Now you can use this in navbar like this
```php
<nav>
<a href="/" class="<?= urlIs('/home.php') ? 
'bg-gray-900 text-white' : 'text-gray-300' ?> rounded-md px-3 py-2 text-sm">HomePage</a>
<a href="/about">About Us</a>
<a href="/contact">Contact Us</a>
</nav>
```
